百度首页 Demo 说明

这是一个模拟百度首页的演示页面，包含以下功能：

  • 顶部导航栏（新闻、hao123、地图、视频等）
  • 百度彩色Logo（使用CSS实现）
  • 交互式搜索框（聚焦效果、悬停效果）
  • 语音搜索和图片搜索图标
  • "百度一下"和"手气不错"按钮
  • 底部链接和信息
  • 响应式设计（支持移动端）

交互功能：

  • 点击搜索框获得聚焦效果
  • 鼠标悬停在导航链接上有颜色变化
  • "更多"菜单悬停显示下拉选项
  • 点击"百度一下"按钮显示搜索提示
  • 点击"手气不错"按钮显示随机搜索提示
