# 百度首页 Demo

这是一个模拟百度首页的演示页面，用于展示前端开发能力和交互设计。

## 功能特性

### 1. 页面结构
- **顶部导航栏**: 包含新闻、hao123、地图、视频等链接
- **百度Logo**: 使用CSS实现的彩色文字Logo
- **搜索区域**: 交互式搜索框和功能按钮
- **底部信息**: 版权信息和相关链接

### 2. 交互功能
- **搜索框聚焦效果**: 获得焦点时显示蓝色边框和阴影
- **悬停效果**: 导航链接、按钮等元素的悬停状态
- **下拉菜单**: "更多"按钮的悬停下拉菜单
- **按钮交互**: "百度一下"和"手气不错"按钮的点击反馈
- **图标功能**: 语音搜索和图片搜索图标的点击提示

### 3. 响应式设计
- 适配桌面端和移动端
- 在小屏幕设备上自动调整布局
- 触摸友好的交互元素

## 技术实现

### HTML结构
- 语义化HTML标签
- 清晰的DOM结构
- 良好的可访问性基础

### CSS样式
- 现代CSS布局（Flexbox）
- CSS变量和过渡动画
- 响应式媒体查询
- 自定义颜色和字体

### JavaScript交互
- 事件监听和处理
- DOM操作
- 模拟搜索功能
- 随机搜索提示

## 使用说明

1. 直接在浏览器中打开 `baidu-demo.html` 文件
2. 体验各种交互功能：
   - 点击搜索框输入文字
   - 鼠标悬停在导航链接上
   - 点击"百度一下"按钮进行搜索
   - 点击"手气不错"获取随机搜索
   - 体验语音和图片搜索图标

## 注意事项

1. 这是一个纯前端的演示页面，不包含后端功能
2. 所有搜索功能都是模拟的，不会实际访问百度服务器
3. 页面设计参考了百度首页的视觉风格，但进行了简化
4. 仅供学习和演示使用
