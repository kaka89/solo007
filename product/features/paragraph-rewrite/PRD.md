# 段落一键重写 — 需求文档

## 用户故事

**作为**一个 WriteFlow 用户，
**我希望**选中一段文字后能一键 AI 重写，
**以便**快速改善表达质量，无需手动逐句修改。

## 场景描述

1. 用户在编辑器中选中一段文字
2. 浮动工具栏出现「重写」按钮
3. 点击后 AI 生成重写版本，保留原意
4. 用户可选择「采用」或「放弃」

## 验收标准

- [ ] 选中文本后 500ms 内出现重写按钮
- [ ] 重写延迟 < 3 秒（P95）
- [ ] 重写结果保留原文核心语义
- [ ] 支持「采用/放弃/重新生成」三种操作
- [ ] 错误时显示友好提示，不阻断编辑流程
- [ ] 移动端触摸选中同样可用

## 非功能需求

| 指标 | 要求 |
|------|------|
| 响应延迟（P95） | < 3 秒 |
| 并发支持 | >= 50 用户同时重写 |
| 错误率 | < 1% |
| 降级策略 | API 超时后 5s 自动重试一次，仍失败则提示用户 |

## 假设验证状态

来源假设 H-001：「用户需要段落一键重写功能」
- 验证方式：邀请 5 位活跃用户内测 Beta，观察 3 天使用频率
- 当前状态：testing

## 竞品参考

- **Jasper AI**：有类似功能但中文支持弱，价格高（$49/mo）
- **Notion AI**：AI 能力是附加功能，写作专注度不如专用工具
- WriteFlow 差异化：聚焦中文写作体验，价格更亲民
